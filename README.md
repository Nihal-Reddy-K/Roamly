Roamly – Full Stack Web Application

Live Site:
https://roamly-p3x8.onrender.com


Project Overview:

Roamly is a full-stack web application inspired by Airbnb where users can explore lodging listings, create their own listings, and leave reviews. The project demonstrates core full-stack development concepts including authentication, database relationships, image uploads, geolocation maps, filtering, and deployment.


Features:

User authentication (signup, login, logout)

Create, edit, and delete property listings

Upload listing images with Cloudinary

Interactive location maps using Mapbox

Review and rating system for listings

Category filters (Beach, Mountains, City, etc.)

Search functionality

Dark mode toggle

Tax price toggle

Responsive UI using Bootstrap

Full deployment with MongoDB Atlas and Render
